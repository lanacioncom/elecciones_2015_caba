# elecciones_2015_caba


## Front End

### Install dependences

```bash
npm install & bower install
```

### Build web app for production

```bash
gulp build
```

### Run server

```bash
gulp server
```

### Deploy build to ftp production

Rename 'keys_ftp.js.example' to 'keys_ftp.js' and put in your keys

```bash
gulp deploy_ftp
```
